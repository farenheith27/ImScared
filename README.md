# Weekly routine
### Monday - Planning day
- Open Notion Dashboard
- Choose a couple of realistic _(eheh)_ objectives for the week
- Define choosen arguments time and point them into calendar
- Create a branch on the repository about the week

### Thursday to Saturday - Output
- Open dedicated branch
- Study some of the arguments
- Get done a micro-output of any kind about what you learned
- Commit -> Push

### Sunday - ~Roll down and cry~ Review
- Merge the weekly branch into main
- Write down a recap ~in Notion~ into a dedicated "journey.md" repository file with:
  - What you learned
  - What you did
  - What is not clear
- Think about next weekly branch title
